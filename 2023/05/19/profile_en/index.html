Project and Research Experience

1. Master’s Thesis: Domain-specific Accelerator for Distributed Deep Learning and Parallel Computing

  • Summary: Designing Roar (A Router Microarchitecture for In-network Allreduce).
  • The allreduce operation collects data from all hosts and then broadcasts the aggregated results back to them. In-network computing can offload allreduce into network devices, which can improve the performance by at least 2x. In this work, we integrate an in-network accelerator in the standard router. The architecture is tailored for implementation on RDMA networks. Only packets requiring aggregation are intercepted by Roar’s unit. Once aggregated, the results are sent back to the router input units and routed to the next hop. This work has been published at the 2023 International Conference on Supercomputing (ICS’23).
